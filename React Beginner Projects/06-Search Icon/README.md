# 🔍 Search Toggle (React Project)

A simple and interactive **Search Toggle UI** built using React. Click the search icon to open an input field and click outside to close it.

---

## 🚀 Features

- Initially shows a **search icon**
- On click:
  - Displays **input field**
  - Changes background color
- Clicking outside:
  - Hides input
  - Resets background

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS
- React Icons

---

## 🧠 Concepts Used

- useState hook
- Conditional rendering
- Event handling
- DOM event targeting
