# 🧮 React Calculator App

A simple and responsive calculator built using **React.js**.
This project performs basic arithmetic operations with a clean UI and interactive buttons.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear input
* 📱 Responsive design
* ⚡ Instant calculation

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3

---


## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/react-js-projects.git
```

2. Navigate to the project:

```
cd 04-Calculator
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
