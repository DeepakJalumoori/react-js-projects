# 📝 Todo List App

A simple and interactive Todo List application built using **React.js**.
This project helps users add and remove tasks easily, making it a great beginner-friendly project to understand React fundamentals.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ⚡ Instant UI updates using React state
* 🎯 Simple and clean user interface

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/DeepakJalumoori/react-js-projects/tree/main/React%20Beginner%20Projects/02-Todo-List
```

2. Navigate to the project folder:

```
cd todo-app
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm run dev
```
