# 🍽️ Meal API Project

A simple React project that fetches and displays seafood meals using an external API. This project demonstrates how to use **Axios**, **React Hooks**, and basic component structuring.

---

## 🚀 Features

* Fetch data from an external API
* Display meals with images and IDs
* Clean and responsive UI
* Uses React Hooks (`useState`, `useEffect`)
* Handles API calls using Axios

---

## 🛠️ Tech Stack

* React.js
* Axios
* CSS

---

## 📡 API Used

* [TheMealDB API](https://www.themealdb.com/api.php)
* Endpoint:

  ```
  https://www.themealdb.com/api/json/v1/1/filter.php?c=Seafood
  ```

---


## ⚙️ Installation & Setup

1. Clone the repository

   ```
   git clone "https://github.com/DeepakJalumoori/react-js-projects/tree/main/React%20Beginner%20Projects/03-Meals%20Api%20Project"
   ```

2. Navigate to the project folder

   ```
   cd meal-api-project
   ```

3. Install dependencies

   ```
   npm install
   ```

4. Start the development server

   ```
   npm start
   ```

---

## 📸 How It Works

* When the component loads, an API request is made using Axios
* The response data is stored in state
* The meals are displayed dynamically using `.map()`

---

## 🧠 Key Concepts Learned

* API fetching with Axios
* React Hooks (`useState`, `useEffect`)
* Rendering lists in React
* Handling asynchronous data
